# Models

The current Aegis MVP intentionally uses deterministic, explainable rules rather than a trained predictive model. Evidence from public environmental and infrastructure sources is converted into an operational risk assessment for qualified human review.

This directory is reserved for future, validated model artifacts and model cards. Any model added here must document its training data, evaluation, calibration, intended use, known limitations, and human-oversight requirements.

See the [AI/ML Improvement Roadmap](../docs/AI_ML_ROADMAP.md) for the planned progression from rules to validated elevated-risk screening.
